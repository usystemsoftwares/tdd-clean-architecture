## Description

Project developed in typescript with clean architecture + tdd for demonstration purposes.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn dev

# watch mode
$ yarn start:dev

# production mode
$ yarn build
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
